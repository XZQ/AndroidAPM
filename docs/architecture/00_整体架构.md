## 三、事件处理流程

```
                        ┌─────────────────────┐
                        │   业务模块           │
                        │  (Memory/Crash/ANR..)│
                        └──────────┬──────────┘
                                   │
                          Apm.emit(module, name,
                                   kind, severity, fields)
                                   │
                                   ▼
                        ┌─────────────────────┐
                        │      Apm.kt         │
                        │  (统一入口 Object)   │
                        └──────────┬──────────┘
                                   │
                          state?.context?.emit(event)
                                   │
                                   ▼
                        ┌─────────────────────┐
                        │    ApmContext.kt     │
                        │  dispatcher.emit()   │
                        └──────────┬──────────┘
                                   │
                                   ▼
                   ┌───────────────────────────────┐
                   │        ApmDispatcher          │
                   │                               │
                   │  executor.submit {            │
                   │    ① 限流检查                  │
                   │       RateLimiter.tryAcquire() │
                   │       (ERROR/FATAL 跳过限流)   │
                   │                               │
                   │    ② 本地存储                  │
                   │       store.append(event)      │
                   │       → FileEventStore         │
                   │         → ring buffer (500行)  │
                   │         → 文件持久化           │
                   │                               │
                   │    ③ 上传                      │
                   │       uploader.upload(event)   │
                   │       → RetryingApmUploader    │
                   │         → 队列缓冲 (500)       │
                   │         → 批量上传 (10/批)     │
                   │         → 指数退避重试         │
                   │         → Http/Logcat uploader │
                   │  }                             │
                   └───────────────────────────────┘
```

## 四、模块生命周期

```
                    应用启动
                       │
            ┌──────────▼──────────┐
            │  Apm.init(app,config)│
            │  synchronized(init)  │
            │  创建 Logger/Store   │
            │  创建 Uploader       │
            │  创建 Dispatcher     │
            │  创建 ApmContext     │
            └──────────┬──────────┘
                       │
            ┌──────────▼──────────┐
            │ Apm.register(module) │
            │                      │
            │  module.onInitialize │◄─── 获取 ApmContext
            │  module.onStart()   ◄├─── 注册后立即启动
            │                      │     访问 Logger/Config 并注册回调/启动线程
            └──────────┬──────────┘
                       │
                  应用运行中
                  (持续采集)
                       │
                       │  Apm.emit() ◄─── 各模块随时上报事件
                       │
            ┌──────────▼──────────┐
            │    Apm.stop()        │
            │                      │
            │  module.onStop()    ◄├─── 停止监控
            │  (每个模块)          │     释放资源/线程
            │                      │
            │  dispatcher.shutdown │
            │  uploader.shutdown  │
            └─────────────────────┘
```

## 五、线程模型

```
┌─────────────────────────────────────────────────────────────┐
│                        线程全景                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Main Thread                                                 │
│  ├── Apm.init() / register()                                │
│  ├── Apm.emit() (非阻塞，提交到 dispatcher 线程)             │
│  └── 各模块 Lifecycle 回调                                   │
│                                                             │
│  apm-dispatcher (SingleThreadExecutor)                       │
│  ├── 限流检查 (RateLimiter)                                  │
│  ├── 本地存储 (FileEventStore.append)                        │
│  └── 上传 (RetryingApmUploader.upload)                       │
│                                                             │
│  apm-upload-retry (SingleThreadExecutor)                     │
│  ├── 从队列取事件 (LinkedBlockingQueue, cap=500)             │
│  ├── 批量聚合 (batchSize=10)                                 │
│  └── 指数退避重试 (base=1s, max=3次)                         │
│                                                             │
│  apm-anr-sigquit (SingleThreadExecutor)                      │
│  └── SIGQUIT 到达后在独立线程执行堆栈采样/分类/上报           │
│                                                             │
│  memory-sampler (ScheduledExecutorService)                   │
│  ├── 定时采集 MemorySnapshot (前台15s/后台60s)               │
│  └── PSS/Heap/ProcStatus 数据收集                            │
│                                                             │
│  apm-anr-watchdog (Thread)                                   │
│  ├── Watchdog 循环检测主线程 tick                            │
│  └── 超时触发 ANR 分析                                      │
│                                                             │
│  apm-stack-sampler (HandlerThread)                           │
│  ├── 定时采样主线程堆栈 (默认10ms间隔)                       │
│  └── 热点方法统计                                            │
│                                                             │
│  apm-io-leak-monitor (Thread, daemon)                        │
│  └── PhantomReference 队列监控 Closeable 泄漏               │
│                                                             │
│  apm-io-fd-monitor (Thread, daemon)                          │
│  └── 定期扫描 /proc/self/fd 统计 FD 数量                    │
│                                                             │
│  apm-leak-check (HandlerThread)                              │
│  └── Activity 泄漏延迟检测 + GC 触发                        │
│                                                             │
│  hprof-dump (ExecutorService)                                │
│  └── 异步 Hprof 文件 dump                                   │
│                                                             │
│  CPU Jiffies Sampler Thread                                  │
│  └── 定时读取 /proc/self/stat 计算 CPU 使用率               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 六、数据模型

```
┌──────────────────────────────────────────────────────┐
│                     ApmEvent                         │
├──────────────────────────────────────────────────────┤
│ module: String        │ "memory" / "crash" / "anr"...│
│ name: String          │ "memory_snapshot" / "java_..."│
│ kind: ApmEventKind    │ METRIC | ALERT | FILE        │
│ severity: ApmSeverity │ DEBUG→INFO→WARN→ERROR→FATAL  │
│ timestamp: Long       │ System.currentTimeMillis()   │
│ processName: String   │ 当前进程名                    │
│ threadName: String    │ 调用线程名（emit 时捕获）     │
│ scene: String?        │ 场景标识                      │
│ foreground: Boolean?  │ 前后台状态                    │
│ fields: Map<String,Any?> │ 业务数据（指标/堆栈/路径） │
│ globalContext: Map    │ 全局上下文（静态+动态）       │
│ extras: Map           │ 扩展键值对                    │
├───────────────────────┴─────────────────────────────┤
│ toLineProtocol(): String                            │
│ "measurement|field1=v1,field2=v2|tag1=t1|timestamp" │
└─────────────────────────────────────────────────────┘
```

## 七、限流与灰度架构

```
┌─────────────────────────────────────────────────────┐
│                  限流 + 灰度 + 动态配置               │
├─────────────────────────────────────────────────────┤
│                                                     │
│  RateLimiter (令牌桶)                                │
│  ├── ConcurrentHashMap<key, TokenBucket>            │
│  ├── key = module + ":" + name                      │
│  ├── maxEventsPerWindow (默认 10/60s)               │
│  ├── CAS 原子操作                                    │
│  └── ERROR/FATAL 级别跳过限流                        │
│                                                     │
│  GrayReleaseController                              │
│  ├── featureFlags: ConcurrentHashMap<String, Bool>  │
│  ├── isEnabled(feature, userId) → 功能开关           │
│  ├── isInSample(userId, sampleRate) → 用户采样      │
│  └── setFeatureOverride(feature, enabled) → 本地覆盖│
│                                                     │
│  DynamicConfigProvider (接口)                        │
│  ├── getBoolean(key, default)                       │
│  ├── getLongValue(key, default)                     │
│  ├── getFloatValue(key, default)                    │
│  └── getString(key, default)                        │
│  (对接 Apollo / Firebase / 自建配置中心)             │
│                                                     │
└─────────────────────────────────────────────────────┘
```
